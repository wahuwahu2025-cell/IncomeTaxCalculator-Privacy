Last updated: 1, March 2026

Anonymous usage analytics

We use New Relic for analytics and performance monitoring.

We do not collect:

Names

Email addresses

Account information

Location data

Contacts

2. How We Use Information

The information collected is used solely to:

Improve app functionality

Monitor performance

Fix bugs

Understand feature usage

We do not sell or share personal data with third parties.

Analytics data may be processed by New Relic in accordance with their privacy practices.

4. Data Retention

Analytics data is retained only as long as necessary for performance monitoring and product improvement.

5. Your Rights
